# ⚡ LeetCode Patterns Trainer (JavaScript)

An interactive, self-contained study tool for learning the ~16 patterns that cover nearly every LeetCode problem. Built ground-up for someone learning the patterns, comprehensive enough to prep for almost any question.

**No build step, no dependencies** — it's a single `index.html`. Double-click it, or serve it locally.

## Run it

Just open the file:

```bash
open index.html
```

Or serve it on localhost:

```bash
python3 -m http.server 8077
# then visit http://localhost:8077/
```

## What's inside

Four tabs:

1. **Patterns** — all 16 pattern cards. Click any for a deep dive: what it is, how to recognize it (trigger words), complexity, a reusable JS template, a worked example, common pitfalls, and a 5-problem practice ladder (easy → hard). Mark patterns "learned" — progress is saved in your browser.
2. **How to Identify** — an ordered decision checklist. Read the problem, walk top to bottom, first match wins. Trains the core skill of picking the right pattern.
3. **Recognition Quiz** — 20 problem descriptions; pick the pattern, get instant feedback and the reasoning.
4. **Study Plan** — the 16 patterns grouped into 4 tiers in a sensible learning order.

## The 16 patterns

Two Pointers · Sliding Window · Hash Map / Set · Binary Search · DFS · BFS · Backtracking · Dynamic Programming · Heap / Priority Queue · Intervals · Linked List · Stack / Monotonic Stack · Trie · Union-Find · Greedy · Bit Manipulation · Topological Sort
